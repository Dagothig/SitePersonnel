Tiny Religion

This game was made for the Ludum Dare 23.
A simple playthrough can be found here.
It's about making your god happy by doing stuff which mostly consists of doing babies (yes, in this game you do babies) and build bridges.
The interface is text-based and instructions to use it can be accessed in-game, there is also a list of the possible commands in this page.

The game was made in C# with the XNA framework and thus, sadly, can only be run under Windows as well as requiring the .net framework and the XNA framework to be installed.
